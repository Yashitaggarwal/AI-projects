# 🛠️ AI Co-Founder & Product Builder

An autonomous AI team designed to take a raw, one-sentence startup idea and turn it into a comprehensive business and technical plan.

## Features
- **Market Intelligence Lead:** Uses web-searching tools to actively look up competitors and market trends based on your idea.
- **Chief Product Officer:** Writes a formal Product Requirements Document (PRD) detailing the target user and MVP features.
- **Lead Systems Architect:** Designs the optimal tech stack required to build the MVP.

## Installation
```bash
pip install -r requirements.txt
```

## Usage
1. Provide your Gemini API key in the `.env` file or directly in the sidebar.
2. Run the app:
```bash
streamlit run app.py
```
