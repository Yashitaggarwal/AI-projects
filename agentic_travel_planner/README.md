# ✈️ Agentic Travel Planner (Upgrade)

An autonomous travel concierge that builds detailed, weather-aware, and logic-driven travel itineraries.

## Features
- **Logistics Agent:** Checks travel logistics and creates realistic day-to-day scheduling.
- **Local Fixer Agent:** Discovers local gems, restaurants, and cultural experiences.
- **Budget & Routing Agent:** Ensures the itinerary is geographically sensible and cost-effective.

## Installation
```bash
pip install -r requirements.txt
```

## Usage
1. Provide your Gemini API key in the `.env` file or directly in the sidebar.
2. Run the app:
```bash
streamlit run app.py
```
